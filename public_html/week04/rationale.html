Subject:

Header:

There was a major change in the header. First of all, I margined the image-header to the top without having any space. Secondly, I changed the layout of the navigation menu with a fixed position. lastly, I added a search bar for the user, which makes it easier to navigate.


Background:

I have changed the background for the entire site with an image. The image looks good and attract the user's eye. I also set the background in a fixed position.


Body:

The body was to wide on the base page, so I had to magrin it and center it to 30px. I also break it down to sections, that way I had the power to style it better, the body looks good and you still can see the background image. The body is set with a static position, so I can scroll up and down using the body only.


Forms:

I have changed the forms from being titled by names and changed it to placeholder title. This way I saved more space and it looks fancy than before.


Images:

The images were totally out of control due to the body changes, so I put them in a different class and change the sizes to fit with new body pages.

Links:

I had to do some changes also with the links especially the header, since I added the search bar, it was hard to fit all of them in one line-block. However, the change was made to make all of them fit on same line.
